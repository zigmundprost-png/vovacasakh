## Визуальный рефреш header и первого экрана

Структуру и тексты не трогаю — обновляю только визуальную систему: типографику, палитру, плотность, форму карточек и кнопок.

### 1. Палитра (`src/styles.css`)
Перевожу токены на палитру по брифу:
- `--background` → #F8F6F2
- `--foreground` → #222222
- `--muted-foreground` → #5F5F5F
- `--card` → #FFFFFF
- `--surface` → светлый нейтральный для иконочных подложек
- `--primary` → #D97732, `--primary-hover` чуть темнее
- `--border` → #E8E2D8
- `--radius` → 0.75rem (более современные мягкие, но не «капсульные» углы)

Убираю тёплое свечение в фоне hero и «коричневые» оттенки во вторичных элементах.

### 2. Типографика
- Подключаю **Inter** (400/500/600/700) через Google Fonts.
- Удаляю Fraunces / serif. `--font-display` = `--font-sans` = Inter.
- H1: 40px mobile → 58px desktop, `font-semibold`, `tracking-tight`, `leading-[1.1]`. Сильный, но не тяжёлый.
- Body/подзаголовок: 17px, `leading-relaxed`, цвет `muted-foreground`.

### 3. Header
- Высота меньше, без двойного вертикального паддинга.
- Лого-марка: маленький квадрат 32px с инициалами «ВК» на тёмном фоне (вместо круга на тёплой подложке).
- Название: 15px semibold, подпись 11px muted.
- Навигация: 14px, цвет muted, hover → foreground. Без подчёркиваний и фоновых хайлайтов.
- CTA в шапке: прямоугольная кнопка с `rounded-lg` (вместо «pill»), без иконки — только текст. Заметная за счёт акцентного цвета, но визуально легче.
- На скролле — лёгкий blur + тонкая нижняя граница.

### 4. Hero
- Убираю радиальный градиент в фоне — чистый фон.
- Больше воздуха: `pt-20 pb-24` на desktop.
- Заголовок: первая строка foreground, вторая — `muted-foreground` (приём для современного, спокойного ощущения), без оранжевой подсветки слов.
- Метка локации сверху без рамки/чипа — просто точка + текст.
- Доверительная строка: 14px, через тонкие разделители «·».
- Кнопки: `rounded-lg`, без иконок. Primary — оранжевая, secondary — белая карточка с тонкой рамкой. Hover у secondary темнит рамку, без заливки.
- Микротекст 13px muted.

### 5. Правый визуальный блок
- Полностью убираю диагональный полосатый паттерн.
- Сетка проще: 2 колонки сверху (квадратные/портретные `aspect-[4/5]` карточки «Фото до» и «Фото после»), снизу широкая `aspect-[16/9]` «Процесс ремонта».
- Карточки: белый фон (`bg-card`), тонкая рамка `border-border`, `rounded-xl`. Без теней, без градиентов.
- Внутри: маленькая нейтральная иконка-чип в углу + подпись внизу. Никаких ярких акцентов.

### 6. Блок доверия
- Карточки на белом фоне с тонкой рамкой (вместо тёплой «коричневой» полосы поверх surface).
- Иконка в нейтральном квадрате со скруглением `rounded-lg`, акцентный цвет только на самой иконке.
- Заголовок 15px semibold, подтекст 13px muted, `leading-relaxed`.
- Hover: рамка темнеет до `foreground/20`.
- Внешняя «полоса» секции теряет цветной фон — становится продолжением общего фона с верхней границей.

### Что НЕ меняю
- Структуру header (лого + nav + CTA), текст навигации, тексты hero, кнопки MAX, sticky mobile CTA, секции-заглушки, route-файл.
- `MAX_LINK` остаётся в `src/lib/links.ts` с тем же placeholder и комментариями.
